## Summary

A 1:1 trading platform for secure transactions. A platform designed for secure transactions between sellers and buyers.

- Responsive website
- UI work using shadcn/ui
- API integration and data state management using Recoil
- Multilingual support using i18n

## Achivement

Based on the client's provided UI, I selected the tech stack through thorough communication with client. The project was carried out with active communication and schedule sharing via Notion and Slack. As the platform had to cater to two types of users, Buyer and Seller, I carefully considered the user flow. I also put significant thought into how to build reusable components efficiently.
